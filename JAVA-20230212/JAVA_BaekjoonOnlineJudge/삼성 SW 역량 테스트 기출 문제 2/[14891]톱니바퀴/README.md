### 정지민
* **구현/재귀**
* 현재 톱니바퀴 기준으로 옆 톱니바퀴 확인 후 현재 톱니바퀴 회전
  * 왼쪽 톱니바퀴 확인
    * 회전할 경우 왼쪽으로 또 확인
    * 회전하지 않을 경우 탐색 멈춤
  * 오른쪽 톱니바퀴 확인
    * 회전할 경우 왼쪽으로 또 확인
    * 회전하지 않을 경우 탐색 멈춤
* n번 톱니바퀴 0번째를 확인, 2의 n승으로 점수 계산
