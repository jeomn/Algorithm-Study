# 거리두기 확인하기(2022-01-25)
### 이지순

---
### 정지민
* **구현**
* 초기 데이터 [1, 1, 1, 1, 1] 세팅
* 전체 칸(5*5)을 돌며 거리두기 확인
* 앉은 사람이 없는 경우(빈 자리/칸막이) continue;
* 앉은 사람이 있으면 거리 확인
  * 4방향으로 상하좌우에 사람이 앉아있는 지 확인
  * 칸막이가 있다면 continue;
  * 비어있다면, 그 너머의 자리까지 확인
  * 대각선으로 확인
    * 대각선 자리를 기준으로 자리 사이가 칸막이인 지 확인
* 거리두기가 되지 않았을 경우 해당 answer를 0으로 처리, break로 다음 대기실 확인
